Quiz CSS débutant

❶

Que signifie CSS ?

CSS est l'abréviation de Cascading Style Sheets (Feuilles de Style en Cascade).

❷

A quoi sert le langage CSS ?

Le CSS est un langage simple qui sert à styler (polices, couleurs, positions, dimensions ...) les documents internet.

❸

Pourquoi utilise-t-on généralement du CSS ?

Le CSS est généralement utilisé pour séparer le contenu (HTML, XHTML, XML) de sa mise en forme. Cela permet une maintenance simplifiée selon qu'il faille en modifier son aspect ou son contenu.

❹

Où est-il conseillé de placer le code CSS ?

Bien que les 4 possibilités évoquées soient possibles, il est vivement déconseillé de placer le code CSS à l'intérieur du HTML dans le body. L'utilisation d'un fichier externe est recommandé.

❺

Dans quoi définit-on les déclarations CSS ?

Un bloc de déclarations CSS est délimitée par des accolades. En son sein, chaque déclaration (couple "propriété : valeur") est ponctuée par un point-virgule.

❻

Dans la règle suivante, quel élément deviendra rouge ? strong#intro {background-color: red;}

LPuisqu'il n'y a pas d'espace entre "strong" et "#intro", cela signifie qu'il s'agit du même élément. Il n'y a pas de notion de contenant et de contenu ici. La règle s'applique à un seul élément : l'élément <strong> dont l'id est "intro".

❼

Quel élément sépare la propriété de sa valeur ?

La propriété font-size est séparée de sa valeur (exprimée ici en unité variable -em) par deux points : font-size : 1em;

❽

Pourquoi est-il d'usage de sauter une ligne entre chaque déclaration ?

Il n'est pas nécessaire de sauter des ligne ou d'indenter votre code. Mais il sera plus facile à lire, à corriger et à débugger

❾

À quoi sert le sélecteur de classe : h2.nomDeClasse ?

Le sélecteur de classe different permet de modifier l'apparence des titres de niveau 2 dont la classe est "different" et uniquement ceux possédant cette classe.

❿

Combien de règles peut contenir un fichier CSS ?

Un fichier CSS n'est limité ni par son poids, ni par le nombre de ses règles. Vous pouvez donc coder de gros fichiers. Mais pensez quand même à les optimiser, ce n'est pas la peine non plus de tenter le Guiness Book du plus long fichier CSS !
